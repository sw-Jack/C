* 선택문 Switch ~ case 문
 - 값에 따라서 실행 내용을 선택하여 실행하는 제어문
 - 공간의 값에 따라서 해당 case 문을 가서 바로 실행되는 형태

 - 형식
    Switch( 공간 ) {
        case  상수1 : 
                실행내용;
         case 상수2 : 
                실행내용;
         case 상수3 : 
                실행내용;
         case 상수4 : 
                실행내용;
         ..........
         defaulat :
                실행내용;
        }
 - case 문의 상수는 정수 타입의 데이터만 사용 가능하다.
 - 공간은 일반적으로 변수를 사용한다.
    - 계산식이 들어갈 수 있다... 최종적인 값의 형태가 정수 타입이어야 한다.
 - default는 if문의 else와 비슷한 역할을 한다.
 - 별다른 제약이 없으면 해당 case 문부터 순차적으로 실행한다.

 - break
    - if 문을 제외한 가장 처음 만나는 중괄호를 벗어나는 역할
    - switch ~ case, while, do ~ while, for 문
